## 🚀 Environment Setup

1. 🕸️ **Neo4j:** Install Neo4j Desktop or access Neo4j Browser.
2. 🌐 **Browser:** Open Neo4j Browser at `http://localhost:7474` (default).
3. 📟 **Query:** Execute Cypher commands in the query editor.
4. 🔍 **Verify:** Use MATCH queries to check results.
